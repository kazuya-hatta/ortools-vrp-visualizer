# Simple OR Tools Visualization (Vehicle Routing)

### Prerequisites
- Python3
- Google Cloud Platform Account

### Setup Environment

```sh
scripts/setup
```

Create `.env` file and place it in the root directory.

```sh
DEBUG=True
GMAPS_KEY=xxxxxx..... # -> Retrieve from GCP
```

### Run Server

```sh
scripts/run
```

### Run Tests

```sh
scripts/test
```
